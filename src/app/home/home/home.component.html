This is a demonstration of an Issue Tracker web application created with Angular and Angular Material Design. It stores data locally. You can click Issues button to view the previously raised issues. You will also see Update icon button near to every entry, by clicking you will be able to update the issue or change the status.Click Create button to add some sample Issues.

    To view issues click on Issues Button.
